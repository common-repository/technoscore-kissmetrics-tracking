== Description ==

Technoscore Kissmetrics Tracking is  best plugin for WordPress to add Kissmetrics Tracking code to header part of Selected/All webpages. 

= More details are as follows:- =
* It adds Kissmetrics Tracking code to particular pages or all pages.

== Installation ==

1. Use WordPress' built-in installer
2. Download the ZIP file from WordPress website
3. Extract the technoscore-kissmetrics-tracking directory to your computer
4. Upload the technoscore-kissmetrics-tracking directory to the /wp-content/plugins/ directory
5. Activate the plugin in the Plugin dashboard
